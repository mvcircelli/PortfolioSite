Python Automated Message Bot

A bot I created for a massively multiplayer online community with over 200 members. Featuring over 500 lines of Python, the bot accesses numerous paginated web APIs via GET and POST HTTP requests and searches for desired game-related information. Upon receipt of the desired information or upon community member command, the bot posts the desired information via message to the community. Creating the bot granted valuable experience in APIs, Python libraries, HTTP requests and more.

Thermal and Vibration Environment Testing

Using Siemens NX's Space Systems Thermal simulation environment and the NASTRAN structural simulation suite, I created and carried out thermal and vibrational testing on a PCB. Using my proficiency in NX, I was able to simulate the thermal loads that the PCB would experience while orbiting planet Earth, as well as simulate the vibrational loads that the PCB would experience during launch into orbit. Afterwards, I verified the results by putting a real PCB through a thermal test using a TVAC chamber, and a vibration test using a shaker table. I then compiled my observations from the real life test into a 17 page report for review of the results.
